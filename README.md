# Natours Tour Bookings Web App

Natours is a tour booking web app where interested tourists or users come to get a chance to tour around the United States. It is a touring web app which has price friendly tours for you and your loved ones.

As a user, you can sign up on the natours website to check out the amazing tour packages which have been listed for your comfort. Also, tour guides will be allocated to interested tourists after registration and payment has been made. The job of the tour guide is to make sure that every member on the tour has an amazing time during the tour.
